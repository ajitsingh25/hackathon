Vulnerability :   Secure Flag

Severity:           HIGH

Description: 

1. Secure Flag is an option that can be set by the application server when sending a new cookie to the user within an HTTP Response.
2. Prevents cookies from being observed by unauthorized parties due to the transmission of a the cookie in clear text.

Remediation:

Set the Secure cookie attribute. This can be set as below.
1. Set the property in the Servlet 3.0 specification application code.
2. Set the property in the <Session-Config> of deployment descriptors (web.xml)
