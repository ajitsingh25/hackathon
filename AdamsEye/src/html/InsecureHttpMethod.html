Vulnerability :   Insecure HTTP methods

Severity:           HIGH

Description:

1. HTTP methods can be used for nefarious purposes if the web server is misconfigured.
2. Cross Site Tracing (XST), a form of cross site scripting using the server's HTTP TRACE method, is examined.
3. Could allow an attacker to modify the files stored on the web server and, in some scenarios, steal the credentials of legitimate users

Remediation:

Disable the below HTTP Methods:

  1. PUT
  2. DELETE
  3. CONNECT
  4. TRACE
