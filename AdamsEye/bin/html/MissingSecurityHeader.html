Vulnerability :    Missing Security Header

Severity:             HIGH

Description: 

1. Security enhancement that prevent any communications from being sent over HTTP to the specified domain and will instead send all communications over HTTPS.
2. Prevents a man-in-the-middle attacker who attempts to intercept traffic from a victim user using an invalid certificate and hopes the user will accept the bad certificate.
 

Remediation:

1. The web server side needs to inject the HSTS (HTTP Strict Transport Security) header.
2. For HTTP sites on the same domain it is not recommended to add a HSTS header but to do a permanent redirect (301 status code) to the HTTPS site.
